## Usage

```bash
ai -p "explain this codebase"
```

The `-p` (or `--print`) flag runs the agent without interactive input and prints the result to stdout.

## Structured output

Get machine-readable JSON for CI pipelines:

```bash
ai -p --json "write tests for auth" > result.json
```

Returns:

```json
{
  "output": "created tests/auth.test.ts...",
  "model": "anthropic/claude-sonnet-4.5",
  "tokens": 2847,
  "cost": 0.02,
  "exitCode": 0,
  "chatId": "abc123",
  "usage": {
    "inputTokens": 1800,
    "outputTokens": 1047
  }
}
```

## Flags

<Properties>
  <Property name="-p, --print" type="boolean" required>
    Enable headless mode.
  </Property>
  <Property name="--json" type="boolean">
    Return structured JSON output.
  </Property>
  <Property name="--force" type="boolean">
    Skip all tool confirmations. Auto-approve everything.
  </Property>
  <Property name="--no-save" type="boolean">
    Don't save this session to chat history.
  </Property>
  <Property name="--timeout" type="seconds">
    Abort after N seconds.
  </Property>
  <Property name="--plan" type="boolean">
    Read-only mode. Analyze without modifying files.
  </Property>
  <Property name="-q, --quiet" type="boolean">
    Suppress stderr status messages.
  </Property>
  <Property name="-r" type="chatId">
    Resume a previous session by ID.
  </Property>
</Properties>

## Exit codes

| Code | Meaning |
| --- | --- |
| `0` | Success |
| `1` | Error |
| `2` | Agent stuck |

## Examples

```bash
ai -p --force "fix all type errors"
ai -p --no-save "what dependencies are outdated?"
git diff | ai -p "review this for bugs"
ai -p -m gpt-5 --force "refactor the database layer"
ai -p --timeout 60 "fix type errors"
```

<Callout type="warning">
  The `--force` flag auto-approves all destructive operations including file deletes and shell commands. Use with caution in CI.
</Callout>
