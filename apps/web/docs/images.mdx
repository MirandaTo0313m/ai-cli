## From file

```bash
ai --image ./screenshot.png "what's in this image?"
```

Supports PNG, JPG, GIF, and WebP formats.

## From clipboard

In interactive mode, press <Kbd>Ctrl+V</Kbd> to paste an image from your clipboard. The AI will see it as part of your next message.
