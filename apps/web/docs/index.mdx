ai-cli is a terminal-native AI agent powered by [Vercel AI Gateway](https://vercel.com/docs/ai-gateway). It connects to 50+ models from xAI, OpenAI, Anthropic, Google, and more through a single API key.

## What it does

<Cards>
  <Card title="File operations" description="Read, write, and edit files with AI guidance" />
  <Card title="Code search" description="Search codebases with regex and fuzzy matching" />
  <Card title="Process control" description="Run commands and manage background processes" />
  <Card title="Image analysis" description="Analyze images from clipboard or file" />
  <Card title="Persistent memory" description="Remember facts across sessions" />
  <Card title="CI integration" description="Structured JSON output for pipelines" />
</Cards>

## Quick start

<Steps>
  <Step>
    ### Install globally

    ```bash
    npm install -g ai-cli
    ```
  </Step>
  <Step>
    ### Set up your API key

    ```bash
    ai init
    ```
  </Step>
  <Step>
    ### Start using it

    ```bash
    ai
    ```
  </Step>
</Steps>
