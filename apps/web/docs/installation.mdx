## Install

```bash
npm install -g ai-cli
```

Or with other package managers:

```bash
bun add -g ai-cli
pnpm add -g ai-cli
yarn global add ai-cli
```

## Setup

<Steps>
  <Step>
    ### Get an API key

    Sign up at the [Vercel AI Gateway dashboard](https://vercel.com/d?to=%2F%5Bteam%5D%2F%7E%2Fai%2Fapi-keys&title=Go+to+AI+Gateway) and create a new key.
  </Step>
  <Step>
    ### Run init

    ```bash
    ai init
    ```

    Paste your API key when prompted. It gets stored in `~/.ai-cli/config.json`.
  </Step>
  <Step>
    ### Verify

    ```bash
    ai "hello"
    ```

    If you see a response, you're ready to go.
  </Step>
</Steps>

## Requirements

<Properties>
  <Property name="Node.js" type="18+" required>
    ai-cli requires Node.js 18 or higher.
  </Property>
  <Property name="API key" type="string" required>
    A Vercel AI Gateway API key for model access.
  </Property>
</Properties>
