## How it works

The AI can save facts using its `memory` tool. These facts persist in `~/.ai-cli/memories.json` and are loaded into every future session as part of the system prompt.

## Saving memories

The AI saves memories automatically when it learns something worth remembering. You can also ask it directly:

```
remember that this project uses bun not npm
```

## Viewing memories

```bash
/memory
```

## Storage

<Callout type="info">
  Memories are stored in `~/.ai-cli/memories.json` and loaded into the system prompt at the start of each session. They persist until you manually remove them.
</Callout>
