## Default model

The default model is `anthropic/claude-sonnet-4.5`. Change it in [configuration](/docs/configuration) or per-request with `-m`.

## Switching models

Use the `-m` flag with any model name or shorthand:

```bash
ai -m gpt-5 "hello"
ai -m sonnet "hello"
ai -m gemini "hello"
ai -m llama "hello"
```

<Callout type="tip">
  Fuzzy matching resolves partial names to the closest available model. Type `son` and it matches `anthropic/claude-sonnet-4.5`.
</Callout>

## Listing models

```bash
ai -l
```

Shows all available models from the gateway.

## In-session switching

Use the `/model` slash command in interactive mode:

```bash
/model gpt-5
/model sonnet
```
