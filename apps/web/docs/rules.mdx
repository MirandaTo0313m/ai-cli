## What are rules?

Rules are custom instructions that guide the AI's behavior. They're defined in `AGENTS.md` files and loaded at the start of each session.

## Global rules

Create `~/.ai-cli/AGENTS.md` for instructions that apply to every session:

```markdown
- always use typescript
- prefer functional patterns
- write tests with vitest
```

## Project rules

Create `AGENTS.md` in your project root for project-specific instructions:

```markdown
- this project uses bun
- components go in src/components/
- use tailwind for styling
```

## Viewing active rules

```bash
/rules
```

<Callout type="tip">
  The AI loads both global and project rules automatically. Project rules take precedence when they conflict with global rules.
</Callout>
