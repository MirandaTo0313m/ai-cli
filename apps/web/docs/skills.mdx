## What are skills?

Skills are installable extensions that give the AI new capabilities. They follow an open standard and can be shared via GitHub repositories.

## Installing

<Steps>
  <Step>
    ### Open the skill browser

    ```bash
    /skills
    ```
  </Step>
  <Step>
    ### Browse or search

    The AI can discover and install skills automatically. Ask it directly:

    ```
    find a skill for deploying to vercel
    ```
  </Step>
</Steps>

## Creating skills

Skills are defined as GitHub repos with a standard format. Any repo that follows the convention can be installed as a skill.

<Callout type="tip">
  Skills are a powerful way to extend ai-cli with domain-specific knowledge and workflows. Think of them as plugins that teach the AI new tricks.
</Callout>
